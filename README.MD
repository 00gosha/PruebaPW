## Feria de Emprendedores

Este proyecto es una aplicacion web desarrollada para gestionar la postulacion de emprendedores a nuestra Feria de Emprendedores.

La aplicacion nos permite registrar postulantes a traves de un formulario el cual tiene validaciones en tiempo real de los datos ingresados. Almacenamos los datos localmente.

## Tecnologias

# HTML
# CSS
# JavaScript
# Node.js
# Express

## Dependencias

# Express

Para instalar la dependencia: npm install express

## Requisitos para ejecutar la aplicacion

# Node.js
# npm

## Instalacion
1.- Descargas o clonar el proyecto
2.- Abrir la terminal del proyecto
3.- Instalar dependencias:
npm install

## Ejecucion del proyecto

node servidor.js

Se ejecuta en el puerto: http://localhost:3000
